# ADM 04, 05 e 07 – Operação e Ativos

Este diretório contém os arquivos JSX base relacionados à operação, gestão de ativos, energia e qualidade operacional do sistema TITAN.

## Escopo
- Cadastro de ativos
- Contratos de energia
- Mapeamento de tags
- Tarifas e bandeiras
- Calibração e drivers
- Eficiência energética
- Insumos operacionais
- Qualidade de dados
- Curvas de performance
- Roteamento operacional
- Estoque e ressuprimento

## Observação
Arquivos aqui são insumos de desenvolvimento e não devem ser utilizados diretamente em produção.
