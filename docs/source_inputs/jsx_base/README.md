# JSX Base – TITAN
Domínio: Águas
Interface: Administrador

Este diretório contém os arquivos JSX originais utilizados como base para o desenvolvimento dos módulos do sistema TITAN.

## Objetivo
- Servir como referência estrutural e funcional
- Apoiar a modelagem dos módulos
- Garantir rastreabilidade entre interface e implementação

## Observação
Os arquivos aqui NÃO devem ser utilizados diretamente em produção.

A implementação oficial ocorre em:

- frontend/src/modules/
- backend/modules/
