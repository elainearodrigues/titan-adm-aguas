# ADM 14–17 – Normativos, Qualidade e Compliance

Este diretório contém os arquivos JSX base relacionados à gestão normativa, controle de qualidade, conformidade regulatória e rastreabilidade do sistema TITAN.

## Escopo
- Métodos analíticos
- Outorgas vigentes
- Pacotes de água
- Portarias regulatórias
- Aplicação por planta
- Normas estaduais (ex: ARSESP)
- Pontos de amostragem
- Renovação de licenças
- Histórico de mudanças
- Versionamento de pacotes
- Plano de amostragem
- Volumes captados
- Cadeia de custódia

## Observação
Arquivos aqui são insumos de desenvolvimento e não devem ser utilizados diretamente em produção.
